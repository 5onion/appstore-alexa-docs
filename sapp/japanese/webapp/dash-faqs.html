よくある質問 (DRS)

DRS はどのように開始しますか？

Amazon Dash Replenishment Serviceへの統合を開始するには、「スタート」ボタンをクリックし、プログラムへ契約をよくお読みになった上で同意され、貴殿/貴社のデバイスおよび再発注のユースケースに関する情報をご提供いただきます。

私のネットに接続したデバイスは DRS に対応していますか？

DRS は直接またはプロキシを通してインターネットに接続できるどんなデバイスにも統合できます。

DRS は Dash ボタンとどう違うのですか？

デバイスに関連していない消費財（スポーツドリンクやペーパータオルなど）では、Dash ボタンが DRS 統合として簡単で独立型のソリューションを提供します。 デバイスに関連している消費財（浄水フィルターやプリンターのカートリッジなど）では、デバイスメーカーは DRS への統合に対し2つのオプションがあります。デバイス内に再発注用の物理ボタンを組み込むか、消費財の使用を測って再発注が自動的に行われることです。

DRS は規模の大きい製造業者のみが対象ですか？

いいえ。当社では大企業から愛好家まで、あらゆる規模のデバイスメーカーと協業することを望んでいます。

Amazon では私のデバイス用の消費財を販売していません。 それでも参加できますか？

はい。 フルフィルメント by Amazon を使用することにより、貴殿/貴社独自の消費財を Amazon で販売できます。 その後 DRS を貴社のネットに接続したデバイスに統合すると、それら消費財を再発注できるようになります。

10行のコードで本当に DRS が導入できるのですか？

はい。 デバイスメーカーは簡単な HTML コンテナと REST API コールの使用により、たった10行のコードで DRS の使用を開始できます。 デバイスメーカーは住所、お支払い方法、請求システムの管理に煩わされることなく、お客様に代わって注文を出すことができます。
