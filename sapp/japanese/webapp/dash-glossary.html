用語集 (DRS)

解説

この項では Amazon Replenishment Service（DRS、Dash Replenishment Service）文書全体で使用されている一般用語を定義します。

一般用語

Amazon 標準識別番号 （ ASIN 、 Amazon Standard Identification Number ）
Amazon 固有の製品用リテールカタログ識別番号。
Amazon SNS
クラウド（aws.amazon.com/sns）からの通知を簡単に設定、操作、および送信できるようにした、AWS 簡易通知サービス（Simple Notification Service）。
compositeId
Amazon が作成した識別子で、特定のデバイス・モデルを指します。 DRS 対応デバイスを作成すると、貴殿/貴社は compositeId を受け取ります。
お客様
デバイスのエンドユーザーで、Amazon にアカウントを持ちます。
Dash Replenishment Service （DRS, 補充サービス）
Amazon の Dash Replenishmentを提供するサービス。これによりネットに接続したデバイスのメーカーがお客様に代わって発注することができます。 混乱をさけるため、お客様とのコミュニケーションでは「Amazon Dash Replenishment」を使用します。
登録解除
デバイスをお客様の Amazon アカウントから切り離す行為。
デバイス
お客様が単体として購入、登録、登録解除、または管理できるハードウェア製品。 ひとつのデバイスが Amazon Dash Replenishmentシステムを通して再発注される消費材を複数もつ場合もあります。
デバイスメーカー
DRS に統合されるデバイスの製造業者。
DRS 設定ページ（DRS Settings Page 、DSP ）
Amazon 上およびデバイスの関連ウェブサイトまたはアプリ内でお客様が利用できるページで、ここでお客様は自動再発注状況や自動的に再発注される ASIN など、Amazon Dash Replenishmentシステムの設定を閲覧・変更できます。
Amazon にログイン（Login with Amazon 、LWA ）
お客様が Amazon 以外のウェブサイトやデバイスから Amazon のアカウントにログインできるシステム。 デバイス、その関連ウェブサイトまたはアプリが LWA からトークンを受け取り、それにより Amazon に発注できます。
コンテンツと端末の管理（Manage Your Content and Devices 、MYCD ）
お客様がデバイスの設定を管理できる Amazon 上のページ（http://www.amazon.com/mycd）。
デバイス第1 ページ（Primary Device Page 、PDP ）
お客様のデバイスの詳細および/または状況を提供する、デバイスの関連ウェブサイトまたはアプリの第1ページ。 PDP は通常、関連ウェブサイトまたはアプリで最初に表示される画面です。
登録
固有のデバイス識別番号を用いて Amazon のお客様アカウントにデバイスを結び付ける行為。 Amazon は 信頼できるDRSデバイス登録状況（登録または未登録）を保持し、デバイスメーカーにその状況を伝達します。
サービス加入
デバイスが特定の ASIN を Amazon に発注することを認可するためにお客様が取る行為。 サービスへの加入にはデバイス登録を済ませてあることが要件です。デバイスが発注できるには登録およびサービスへの加入が必要です。 Amazon は信頼できる DRSデバイス登録ス状況（登録または未登録）を保持し、デバイスメーカーに Amazon SNS 通知を用いてその状況を伝達します。
サービス加入の解除
デバイスが Amazon に発注する認可を取り消すためにお客様が取る行為。
WebView
デバイスの関連ウェブサイトまたはアプリ内でお客様に表示される HTML コンテナ。この中で Amazon は DRS のセットアップおよび設定コンテンツを提供します。
