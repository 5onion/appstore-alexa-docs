お客様体験（CX）ガイドライン (DRS)

概要

この項ではお客様に優れた自動発注の体験を確実にお届けすることを目標とし、典型的なデバイスのライフサイクルで各タッチポイントにおけるデバイスメーカー向けのガイドラインを提供します。

デバイスの購入

貴殿/貴社のデバイスにおける自動再発注能力に関して関心を呼び起こすには、Amazon および貴殿/貴社のウェブサイト上の製品詳細ページに DRS との互換性を記述しておかなければなりません。 貴殿/貴社はデバイスの梱包内に紙面の折り込みを入れるか、またはデバイスにステッカーを貼っておくことにより、お客様に DRS への登録案内を行ってください。

サービス登録

貴殿/貴社のお客様にはデバイス設定手順の一環でデバイスを DRS に登録し、DRS にサービスへの加入を行うオプションの表示がなければなりません。 これはデバイスがその意図した使用の前（例えばチュートリアル完了前、「お客様のデバイスは今よりご利用いただけます」に類するメッセージより前）に済ませていなければなりません。

登録およびサービスへの加入はWebViewを使用してお客様に表示され、ページコンテンツは Amazon が管理しています。 Amazon では以下の画面を表示します。

  • ティザーページ - DRS およびそのメリットの説明、そして登録への呼びかけ。 お客様は自動再発注にサインアップするか、辞退しデバイス設定に戻るか、のどちらかを選択できます。
  • Amazon にログイン（ LWA ） - お客様は Amazon の認証情報を入力し、Amazon が DRS 発注詳細および状況についての情報をデバイスメーカーと共有することを許可するよう依頼されます。
  • ASIN の選択 - お客様はデバイスが対象となる ASIN 一覧から再発注する特定の ASIN を選択します。
  • 詳細の確定 - お客様は選択した ASIN、お支払い方法、および発送先住所を確定します。

貴殿/貴社のお客様が登録およびサービスへの加入を完了する前にデバイスの設定手順を中断する場合、登録およびサービスへの加入手順はお客様がデバイス設定手順に復帰したときに表示される必要があります。

お客様がデバイスを初めて設定する際に貴殿/貴社からようこそEメールを送付する場合、そのEメールは DRS への加入を勧める呼びかけを含むことが理想的です。

継続的使用

貴殿/貴社の製品紹介のページ（PDP）には、お客様に DRS への加入設定を管理するオプションを際立たせて表示する必要があります（Amazon Dash Replenishment Serviceへの入り口は PDP から最大0～1回のタップで到達できるように設定）。 この入り口がWebViewで DRS 設定ページ（DSP）を起動します。

DSP は Amazon が管理し、次の情報を提供します。 * お客様のサービス加入状況および加入/解除オプション。 * ASIN 詳細（名前、価格など）および代替の ASIN を選択するオプション（該当する場合）。 * 発送および請求書の詳細（変更オプションあり） * このデバイスからの過去の注文。

デバイスが登録済みでない場合、Amazon では DSP WebViewを呼び出す際に、上記の記述の通りに登録およびサービス加入手順を表示します。

Amazon ではお客様が自分の Amazon アカウントからデバイスを登録解除する機能を提供します。 登録解除があった場合、Amazon は状況の変更を SNS 通知を使用して貴殿/貴社に伝達します。

貴殿/貴社が自動再補充を中止または妨げるようなアクションへのオプションを提供する場合、適切な API コールによりその旨を Amazon に通知する必要があります。 例えばお客様が自分のアカウントからデバイスを消去、削除、または切り離した場合、貴殿/貴社は Amazon に通知しなければなりません。

再発注および在庫管理

再発注および配送の体験を最適化するため、関連情報が Amazon およびデバイスメーカー間で交換され、注文品が予定通りに確実に配送されるようにします。

貴殿/貴社のデバイスは最低一日に一回、消費に関する情報（「前回のメッセージから250gの消費」など）、消費財の残り数量（「在庫残1kg」など）、 および消費財の在庫が0に達する予測日（「0％在庫予測日2015年11月3日」など）を記載したステータスメッセージを Amazon に送信しなければなりません。 詳細はSlotStatus エンドポイントをご覧ください。

貴殿/貴社のデバイスは要求されたお客様への配送日より少なくとも7日前に発注しなければなりません。 これにより納期を確実に守り、Amazon はお客様の要望により注文を取り消すオプションを提供することができます。

商品の発送後、Amazon は貴殿/貴社に予定配送日と数量に関する詳細（「1.2kgを2015年11月3日に配送予定」など）を提供します。

デバイスが発注したオーダーをお客様が取り消した場合、貴殿/貴社は通知を受け取ります。 取り消しが記録され、デバイスが5日後に再発注する旨の通知がお客様に送信されることをご確認ください。

お客様が2回目の発注を取り消した場合、貴殿/貴社は通知を受信します。 貴殿/貴社はこの2度目の取り消しにすぐに対応し、消費財の数量が100％に戻るまで再発注がない旨をお客様に通知する必要があります。

通知およびEメール

Amazon では一貫したお客様体験を提供する目的で、DRS 関連のお客様への通信用に既存のEメール基盤を使用します。 お客様は注文が発信されたとき、発送があったとき、または DRS 加入の設定に重要な変更があったとき（例えばサービスへの加入時）にEメールを受け取ります。 重複を避けるため、貴殿/貴社は DRS の注文/状況についてお客様に通信しないでください。

消費財の配送

貴殿/貴社がデバイス関連の消費財も製造されている場合、DRS を介した注文用に1回分の補充単位を用意されると理想的です。 こうすればお客様は自宅に在庫を保管する必要がなくなります。 1回分の補充単位が実用的でない場合、1カ月以内に使い切れる量の補充パックを提供してください。

お客様のコンテンツとデバイスの管理（MYCD）

お客様はお客様のコンテンツと端末の管理を通して、ネットに接続したデバイスに関する情報にアクセスできます。 具体的にお客様ができることは以下をご覧ください。

  • デバイスを DRS から登録解除
  • DRS 設定の管理（自動補充をオフにする、選択した製品を変更する、支払い/発送詳細を変更するなど）
  • デバイスの識別番号を確認

カスタマーサービス

Amazon は DRS を通したあらゆる注文に関するお問い合わせ（発送の追跡、支払いに関する質問など）へのカスタマーサービスを提供します。

貴殿/貴社にはデバイス設定、Wi-Fi 接続性、DRS の範囲外のサービスへの接続性、製品のハードウェア関連の欠陥など、その他のあらゆるカスタマーサービスのお問い合わせをご担当いただきます。

DRS 発注関連のお問い合わせを貴殿/貴社がお客様より受け取られた場合、以下の手順に沿ってお客様を Amazon にお取り次ぎください。

  • E メール ： Amazon のセルフサービス「お問い合わせ」ページへの以下のリンクを記載したメールをお客様に送付してください。

電話 ： デバイスメーカーのカスタマーサービス係員は Amazon のカスタマーサービス係員にコールを転送します。 Amazon ではデバイスメーカーがデバイスに関する自動補充サービスを開始するより前に、Amazon のカスタマーサービス係員にコールを転送する手順についての詳細をデバイスメーカーにお教えいたします。
