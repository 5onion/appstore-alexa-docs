LWA セキュリティプロフィールを作成 (DRS)

概要

Amazon Dash Replenishment Service（DRS、補充サービス）にアクセスするには、貴殿/貴社の製品がアクセストークンによるAmazon のログイン（LWA）を取得する必要があります。これにより貴殿/貴社の製品はお客様に代わって API にアクセスすることができます。

LWA トークン取得への第一歩は、貴殿/貴社のデバイスにセキュリティプロフィールを作成することです。 この手順は以下の説明でご覧いただけます。

LWA セキュリティプロフィールを作成する

Amazon セキュリティプロフィールの作成方法は以下をご覧ください。

  1. Amazon デベロッパーポータルにログインします。
  2. アプリ＆サービスタブをクリックし、次に Amazon でログインをクリックします。
  3. Create a New Security Profile をクリック。
  4. 次の情報を入力します：
    • Security Profile Name - 貴殿/貴社のアプリの名称
    • Security Profile Description - 貴殿/貴社のアプリの記述
    • Consent Privacy Notice URL
    • Consent Logo Image
  5. Save をクリック。

貴殿/貴社のセキュリティプロフィール情報を取得

貴殿/貴社のセキュリティプロフィールを作成した後、以下のステップに従いデバイス登録の完成に必要な情報を収集してください。

  1. 保存 をクリックすると、画面はセキュリティプロフィール一覧 に切り替わります。 使用したいセキュリティプロフィールの隣りのギアをクリックし、「Security Profile」を選択します。

  2. 貴殿/貴社のSecurity Profile ID およびClient ID をメモしてください。

  3. https://developer.amazon.com/mycid.html に移動し、貴殿/貴社のVendor ID をメモしてください。

アドバイス： 貴殿/貴社のSecurity Profile ID、Client ID、Vendor IDをメモしておくのはデバイスの登録時に必要となるからです。
