デバイス管理および設定 (DRS)

お客様のデバイス管理

Amazon のお客様は Amazon 上の「コンテンツと端末の管理」でコンテンツやデバイスを管理できます。 このページでお客様は随時、DRS が有効になったデバイスを登録解除できます。当初デバイスメーカーの体験を使用して登録されたものも含みます。

デバイスが Amazon で登録解除されると、登録解除の通知が SNS に送信されます。 詳細は通知メッセージの項をご覧ください。 デバイスを登録解除した後は、それ以降の API コールはエラーとなります。

貴殿/貴社のプラットフォーム/アプリケーションがお客様の登録を管理している場合、 デバイスに有効なアクセストークンを用いて登録解除 API を呼び出し、デバイスを DRS から登録解除できます。

DRS 設定ページ

DRS 設定ページ（DRS Settings page、DSP）では、Amazon のお客様が DRS対応の製品に関する設定を操作します。 このページは次の URL を使用することにより、貴殿/貴社のアプリまたはウェブインターフェースでご利用いただけます。

https://drs-web.amazon.com/settings?access_token=foobar&exitUri=https://amazon.com

お客様がこのページを閲覧するには Amazon による認証が必要です。 お客様が最近ユーザーエージェントで Amazon.com にログインしていなかった場合、ログイン手順を体験します。 これにより貴殿/貴社のアプリケーションWebViewは中断されません。

Dash 設定ページ終了ボタン

URLパラメータ exitURI= の使用により、貴殿/貴社のアプリケーションは DSP を終了時に貴殿/貴社のウェブサイトやアプリケーションに戻るパスを提供するための柔軟な方法が得られます。

https://drs-web.amazon.com/settings?access_token=foobar&exitUri=https://amazon.com

URL内に exitURI= パラメータを含めた場合、Dash 設定ページ上に終了ボタンが表示されます。 終了ボタンをクリックすると、貴殿/貴社が URL パラメータに指定した URI にリンクされます。 上記 URL の例では、終了ボタンが表示され、お客様を https://amazon.com にリンクします。 「終了」ボタンにリンクしたい URI に対するスキームの提供が求められます。

URL に exitURI= パラメータを含めない場合、終了ボタンは表示されません。 これは終了ボタンを含むカスタム WebView をご利用になりたい場合に便利です。
